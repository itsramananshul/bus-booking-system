A simple **Bus Booking System** built with Python.  
This program allows users to view buses, book seats with unique booking IDs, cancel bookings, and save data for persistent storage.  

---

## Features

- View available buses and seats
- Book a seat with a **unique booking ID**
- **Persistent data storage** using a file
- Smooth, looping menu for continuous use
- Input validation and error handling
